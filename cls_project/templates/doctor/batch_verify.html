Batch Signature Verification

Efficiently verify multiple CLS signatures at once — designed for IoT telemetry scenarios where many devices sign sensor readings.

Batch Verify Algorithm

Choose random weights ri ← Z*q, then verify:

(Σ ri·σi) · P = Σ ri · (Ti + h1i · (Ri + H₁(RIDi,Ri)·P_pub + h2i·PKi))

LHS uses a single scalar multiplication regardless of batch size. On failure, individual verification isolates bad signatures.
